# electron-fix

> Specific area "throw new Error('Electron failed to install correctly, please delete node_modules/electron and try installing again')"

## Installing

Using npm:

```bash
$ npm install @elmeet/electron-fix -g
```

Start:

```bash
$ electron-fix start
```

or `package.json` add:
```
"scripts": {
  "fix": "electron-fix start"
}
```
```bash
$ npm run fix
```
